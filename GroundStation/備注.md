# GroundStation (地面接收站) 開發備註與進度

## 📌 專案簡介
`GroundStation` 為地面接收系統，主要負責實時接收火箭發射後傳回的 LoRa 遙測封包，將其解包、校驗後，並透過序列埠（Serial Port）傳送至上位機界面，進行飛行軌跡、速度、高度與 GPS 定位的實時繪製與記錄。

## 🛠️ 硬體/軟體設計要點
* **核心 RF 與天線配置**：
  * 地面站 LoRa 接收端使用與火箭端對應的無線通訊晶片（E22 / E80 系列模組）。
  * 必須配置高增益定向天線（例如八木天線），並確保同軸電纜損耗降到最低，以保證發射至數公里高空時仍能流暢通信。
* **上位機與數據儲存**：
  * 開發實時繪圖界面（Dashboard），能呈現 3D 姿態模擬、高度曲線與地圖 GPS 位置。
  * 所有收到的原始數據必須實時寫入日誌檔案（.csv 或 .txt），作為發射後飛行分析的依據。
