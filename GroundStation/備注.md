# GroundStation (地面接收站) 開發備註與進度

## 📌 專案簡介
`GroundStation` 為地面接收站專案。

> ⚠️ **預算與開發策略重大調整**：
> * **硬體完美複用**：為極大化節約開發與生產成本，原定獨立設計的地面接收站 PCB 板**全面取消送印打樣**。
> * **全功能替代方案**：我們將**直接使用一套完整組裝的 `BigAvionic` 四層疊板**作為地面站接收硬體！
>   * 由於 `BigAvionic` 本身已集成高性能 STM32F4 主控晶片、E80 系列 LoRa 遙測通訊模組，以及 USB 串口傳輸界面，其硬體功能已完全覆蓋地面接收與數據回傳的全部需求！
>   * 組員無須重複畫板與焊接地面站，直接利用相同硬體燒錄接收站韌體即可工作！
> * **本目錄功能變更**：本目錄下的舊版 KiCad 專案僅保留作為**技術備份與存檔**。
