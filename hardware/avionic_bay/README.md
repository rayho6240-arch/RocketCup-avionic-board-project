# 📐 Avionic Bay (航電艙機械與結構設計)

這裡存放我們火箭航電艙（Avionics Bay）的 3D CAD 設計模型、機構設計、實體裝配說明以及 3D 列印 STL 檔案。

---

## 📂 檔案結構建議 (Directory Structure)

您可以將機械設計相關檔案存放在此：
* `CAD/`：存放 Fusion 360 / SolidWorks 的原始 `.step` 或 `.f3d` 零件與裝配體模型。
* `STL/`：存放用於 3D 列印的 `.stl` 或是 `.gcode` 切片檔（例如防震疊板固定架、天線底座）。
* `Drawings/`：存放工程圖、開孔尺寸圖（特別是氣壓計排氣開孔尺寸）。

## 🔍 航電艙物理設計核心注意事項 (Avionics Bay Best Practices)

為確保感測器與射頻通訊能正常工作，請在設計航電艙實體結構時，務必遵循以下航太工程標準：

### 1. 氣壓計靜壓艙與排氣孔設計 (Static Port Venting)
* **目的**：`BMP390` 氣壓計必須測量火箭艙外的「靜態氣壓」才能換算精確高度。如果艙體全密封，高度計將完全失效；如果排氣孔設計不當，高速飛行時的「動壓」會導致高度計讀數嚴重偏低或誤判頂點！
* **設計黃金法則**：
  1. 在航電艙外側的圓柱面上，精確鑽設 **3 到 4 個均勻分布的靜壓孔 (Static Ports)**（直徑通常在 1.5mm 至 3.0mm 之間，具體取決於艙體容積）。
  2. 靜壓孔內部可黏貼細緻的「防風海綿 (Venting Foam)」，阻擋外部高速亂流與強光直射氣壓計。
  3. **機艙避光設計 (Light Shielding)**：氣壓計內部的矽半導體對光線敏感，強烈陽光直射會導致高度計讀數出現數十公尺的階躍誤差！請確保航電艙內部結構具備良好的避光性（可用黑色膠帶或遮光隔板）。

### 2. 射頻天線固定與接地 (RF Antenna Mounting)
* **通訊阻礙防範**：若火箭艙體使用碳纖維（Carbon Fiber）或金屬材質，這會形成電磁屏蔽（法拉第籠），阻擋無線電波！
  * **解法**：航電艙段的艙體外殼建議統一使用**玻纖（Fiberglass）**、**紙管**或 **3D 列印 (PLA/ABS/PETG)** 等非導電材質。
  * **天線擺放**：LoRa 天線應垂直架設，並盡量遠離大片金屬結構與高頻噪聲源（如 DCDC 晶片）。

### 3. 電池抗震與固定 (Battery Ruggedization)
* 火箭發射時將面臨劇烈的震動與高達數十個 G 的瞬時衝擊（Shock）。
* 🔋 **電池固定法則**：
  * **絕不能僅用雙面膠或魔鬼氈固定電池**！必須設計實體的**電池盒 (Battery Holder)**，配合束帶或機械鎖緊。
  * 電池接頭（如 JST, XT30）必須設有**防脫落卡扣或絕緣膠帶纏繞**，防止飛行中因震動斷電導致航電系統停擺！
