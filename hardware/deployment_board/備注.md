# Deployment board (開傘與部署板) 開發備註與進度

## 📌 專案簡介
`Deployment board` 主要負責火箭的降落開傘回收（Recovery/Deployment）與火工品（Pyrotechnic charges）的安全引爆控制。該板接收主航電板的氣壓/加速度指令，在達到頂點（Apogee）或特定高度時，精確觸發開傘與火工品控制電路。

## 🛠️ 硬體/軟體設計要點
* **電源轉換與電驅動**：
  * 板載配置 `LM2596S-12-NOPB` 開關電源降壓晶片，用於處理高壓電池輸入，提供充足的驅動功率。
  * 電源迴路需有足夠的過流防護設計，避免開傘瞬間的大電流導致主控 MCU 重啟。
* **安全機制 (Very Critical!)**：
  * **硬體保險 (Arming Switch)**：觸發引爆的電路上必須設計實體保險開關，在起飛前手動上電，防止地面誤觸發。
  * **光耦隔離**：控制訊號與高電壓驅動電路之間必須有光耦隔離設計，防止噪聲反向灌入 MCU 主控。
  * **狀態檢測**：引爆迴路兩端必須有檢測機制，判斷引信是否完好（Continuity Test）。
